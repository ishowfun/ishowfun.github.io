社区版禅道git使用

Interception
2018-07-27
禅道

我们的项目源码是托管在内部的TFS2015的服务器上的，用的是git库，TFS整体来说功能还是蛮强大的，然后使用下来发现还是不太适合我们这种小团队进行项目管理，我个人曾经在团队中使用过禅道，对禅道的管理模型相对了解更多点，于是在尝试了TFS的任务管理后还是放弃了，转向使用禅道任务管理。

禅道官网下载了10.1的windows一键安装版，并安装部署，再进行一系列的基础配置和内容添加，禅道便运行起来了。

10.1版本相对之前的版本并没有太大的改动，所以用起来还是蛮顺手的，社区版对比专业版少了一些功能，比如git集成，一些报表等，而且专业版价格咨询了一下，一个成员360人民币的终身授权，还是很良心的价格，怎奈目前的专业版没有兼容社区版10.1的版本出来，而我又想让禅道和git库集成起来，咋办？自己做呗。

禅道社区版本身是开源产品，而且那些付费内容的接口也是开放着的，只不过没有实现页面而已，可以在后台的二次开发API里找到和git相关的接口，另外，在xampp/zentao/bin/下面有个syncgit.bat是专门用来同步git和禅道任务关联的。

我的需求其实就是进行同步。
